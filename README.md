# Educational Content Management System

This repository contains the LMS frontend and related assets.

Frontend: educational-content-management-system-263732-263747/lms_frontend

Highlights:
- Supabase PKCE auth (REACT_APP_SUPABASE_URL/REACT_APP_SUPABASE_KEY)
- Role-based dashboards and protected routes
- Learning paths/courses/lessons browsing with video playback
- Lesson completion tracking (course_progress)
- Admin dashboards and CRUD for paths, courses, lessons
- Ocean Professional theme

See lms_frontend/README.md for environment variables, routes, and setup details.
